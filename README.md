### 果酱餐饮小程序

果酱餐饮是为餐饮商户和个人提供的支持在线充值、买单服务和会员管理的平台。

### 效果截图

![果酱餐饮](https://cdn.guojiang.club/catering.jpg)

### 功能列表

- 会员管理
- 促销活动管理
- 优惠券
- 在线核销
- 在线充值
- 积分商城
- 订单管理

### 安装

```shell
1. npm install 
2. npm run dev (编译)
3. gulp (编译less)
4. npm run build (打包)
```
